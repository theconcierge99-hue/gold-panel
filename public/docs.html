Overview & pricing

Executive Lounge is a private intelligence terminal: live market wire, creator signals, and paid Concierge AI. The UI is English; Concierge mirrors the user’s language when clear from their messages.

Reader unlock fees split 80% creator / 20% merchant (monthly off-chain). Publish fee is 100% merchant.

Architecture

  • UI — executive-lounge.html + x402-pay.mjs client bundle
  • APIs — Vercel Edge functions under /api/*
  • AI — Google Gemini
  • Payments — x402 v2 via PayAI facilitator (on-chain settlement)
  • Storage — Redis/KV for signals, ledger, and Concierge lounge memory

API reference

Free

  • GET /api/market — headline feed + ticks
  • GET /api/x402-config — public payment config
  • GET /.well-known/x402 — x402 discovery fan-out
  • GET /openapi.json — OpenAPI + payment metadata

Paid (POST, x402)

  • POST /api/news-open — unlock article URL
  • POST /api/concierge — AI chat / enhance / image
  • POST /api/signal-publish — publish signal (1 USDC)
  • POST /api/signal-open — unlock signal body (0.1 USDC)

Without payment, paid routes return 402 with PAYMENT-REQUIRED. GET on paid routes also returns 402 for marketplace probes.

x402 payments

USDC on Base and optionally Solana. Facilitator: PayAI (facilitator.payai.network).

  1. Client receives 402 + payment requirements
  2. Wallet signs USDC transfer
  3. Client retries with PAYMENT-SIGNATURE
  4. Server verifies and settles; response includes PAYMENT-RESPONSE with transaction hash

Set X402_EVM_PAY_TO and/or X402_SOL_PAY_TO in your host environment. Never commit keys or RPC URLs with embedded secrets.

x402scan listing

List this server on x402scan.com for public discovery and transaction indexing.

  • Register your production origin (fan-out discovers all four paid routes)
  • If batch registration fails, register each /api/... URL separately
  • Complete a real micropayment so settlements appear on-chain

Concierge AI

Modes: chat, enhance (signal editor), image (analysis + optional visual).

Uses live market data, general-knowledge feeds, and lounge memory (recent headlines + creator signals).

Language: defaults to English; replies in Indonesian or other languages when the user’s message (or recent user turns for short replies) is clearly in that language.

Creator signals

Publish stores a signal in KV and surfaces it on the market feed. Unlock shows the full summary in a modal.

Requires KV_REST_API_URL and KV_REST_API_TOKEN (or Upstash equivalents) in production.

Security

  • Never commit API keys, KV tokens, private keys, or RPC URLs with keys
  • Do not paste full /api/x402-config responses in public support threads
  • Restrict Gemini keys by HTTP referrer in Google AI Studio
  • Rotate any credential that was exposed

Extended markdown docs for operators live in the repository docs/ folder (same content, security-reviewed).

Deployment

Deploy via Git → Vercel on the correct team and project. Avoid vercel deploy from a personal account that lacks production env vars.

After deploy, verify: site loads, /api/x402-config shows enabled: true, and /api/concierge returns 402 without payment.
