SDK

@conc-exe/agent

TypeScript client for agents: discover Concierge pay-per-call routes, then call intel / security desks with x402 (PAYMENT-SIGNATURE) or TCX credits. No Concierge API keys.

Status: 0.1.0 — monorepo packages/agent · publish with npm run agent:build

Sister SDK: @conc-exe/token-x402 — for merchants monetizing their own APIs (different audience).

Install

npm install @conc-exe/agent
# monorepo: npm run agent:build

Quick start

import { createConciergeAgent, PaymentRequiredError } from "@conc-exe/agent";

const agent = createConciergeAgent({
  origin: "https://conc-exe.xyz",
  // paymentSignature / creditsWallet / settlePayment …
});

const snap = await agent.discover();
console.log(snap.catalog.length, snap.mcpUrl);

try {
  const { body } = await agent.intel.verdict({ message: "DeFi outlook on Solana" });
  console.log(body);
} catch (e) {
  if (e instanceof PaymentRequiredError) {
    console.log(agent.payCurl("intel-verdict", { message: "DeFi outlook on Solana" }));
  }
}

Settle hook

Wire a wallet once; unpaid calls get a 402 challenge, then the client retries:

const agent = createConciergeAgent({
  async settlePayment({ accepts, paymentRequiredHeader }) {
    return await yourWallet.signX402(paymentRequiredHeader, accepts);
  },
});

const { body } = await agent.call("intel-macro", {});

Without an in-process wallet, use pay.sh: agent.payCurl("intel-macro").
