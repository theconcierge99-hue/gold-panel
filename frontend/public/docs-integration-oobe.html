OOBE Protocol

OOBE Protocol powers autonomous AI agents on Solana via the Synapse Agent Protocol (SAP). Concierge is an intel provider — pay-per-call desk verdict and Meteora DLMM data before your agent executes on-chain actions. Settlement uses x402 USDC (PayAI facilitator). No Concierge API key required.

Status: Live — SAP manifest, agt_ ↔ SAP linking, OOBE settlement adapter

How it works

  1. Your OOBE agent (Synapse Studio or SAP CLI) needs market intel before a swap, LP, or stake action.
  2. Call a Concierge tool — concierge:intel-verdict or concierge:intel-meteora — from the SAP manifest.
  3. Agent wallet pays $0.10 USDC per call via x402 (PAYMENT-SIGNATURE header, PayAI facilitator).
  4. Branch on verdict.signal (snipe · watch · avoid) or Meteora pool data, then run your on-chain action.

Concierge uses PayAI/Dexter facilitator x402 today — not SAP escrow headers. Use pay curl or SAP CLI with --endpoint override (see below).

SAP CLI

synapse-sap tools manifest validate \
  https://conc-exe.xyz/distribution/oobe/sap-tools-manifest.json

synapse-sap x402 call <AGENT_WALLET> concierge:intel-verdict \
  --args '{"message":"SOL desk","includeInsider":true}' \

  --save out/verdict.json

Publish to SAP (merchant, one-off)

Register Concierge on-chain as an intel provider — run off Vercel with your merchant wallet (X402_SOL_PAY_TO):

synapse-sap env init --cluster mainnet-beta
synapse-sap tools manifest validate distribution/oobe/sap-tools-manifest.json
synapse-sap agent register --manifest distribution/oobe/sap-tools-manifest.json
synapse-sap tools publish distribution/oobe/sap-tools-manifest.json

Repo path: distribution/oobe/README.md · verify: npm run oobe:verify

OOBE settlement adapter (alternative to PayAI)

When PAYMENT-SIGNATURE is absent, Concierge accepts a confirmed Solana USDC settlement tx from SAP escrow flows:

HeaderValue
X-OOBE-SETTLEMENT-TXSolana transaction signature (recommended)
X-PAYMENT-SETTLEMENTAlias
X-PAYMENT-SIGSettlement tx when OOBE escrow headers are also present

Server verifies USDC credit to X402_SOL_PAY_TO via SOLANA_RPC_URL. Env: OOBE_SAP_X402_ENABLED=1 (default on). Disable with 0.

curl -X POST https://conc-exe.xyz/api/concierge-intel-verdict \
  -H "Content-Type: application/json" \
  -H "X-OOBE-SETTLEMENT-TX: <confirmed-tx-sig>" \
  -d '{"message":"SOL desk","includeInsider":true}'
