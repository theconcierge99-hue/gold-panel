RWA flow

Each published signal is an intelligence certificate (off-chain hash + optional Metaplex NFT on Solana to the creator wallet).

  1. Pay 1 USDC publish fee
  2. Signal appears on feed with ⬡ RWA chip
  3. Creator mints NFT in Phantom (~0.03 SOL gas)
  4. Readers pay 0.1 USDC to unlock full text

Supporting routes: POST /api/lounge-rwa-record-mint, POST /api/solana-rpc-send.

Concierge enhance

Signal authors use POST /api/concierge with mode: "enhance" inside Create Signal (0.1 USDC) — see Concierge Chat.
