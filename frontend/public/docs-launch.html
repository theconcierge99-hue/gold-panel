TCX Launch Playbook

Pre-launch and launch-day env snapshots — paste to Vercel, redeploy, no code changes.

Now: USDC only · TCX shows “Coming soon” in pay modal.

Launch day: set SOON_TOKEN_MINT from pump.fun → all routes accept TCX with holder discount.

Verify: npm run launch:verify -- --phase=pre or --phase=post --live

Env snapshots (repo)

  • config/launch/pre-launch.env.snapshot — use now
  • config/launch/post-launch.env.snapshot — use on token launch day

Post-launch keys

SOON_TOKEN_MINT=<mint_from_pump_fun>
SOON_RESOURCE_KINDS=all
SOON_TOKEN_DISCOUNT_PERCENT=30
SOON_USDC_RATE=0.00008
TOKEN_PAY_DEFAULT_MERCHANT=soon
SOON_HOLDER_FREE_TIER_ENABLED=true
