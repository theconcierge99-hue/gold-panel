Integrations

Metaplex

Creator signals on Executive Lounge can mint optional RWA intelligence NFTs on Solana via Metaplex Token Metadata — separate from x402 USDC payments (creator pays SOL gas in Phantom).

Status: Live (browser mint via Phantom)

Publish fee: 1 USDC via x402 (/api/lounge-signal-publish)

NFT mint: Creator-paid SOL gas — not included in x402 price

Client bundle: public/js/mint-signal.mjs

Flow

Publish signal

Pay 1 USDC x402 fee → signal stored in KV with RWA metadata hash.

Mint in Phantom

Browser calls Metaplex via mint-signal.mjs — NFT to creator wallet.

Record mint

POST /api/lounge-rwa-record-mint links mint address to signal id.
